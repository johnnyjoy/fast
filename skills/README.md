# Skills

Cursor agent skills for **people using Fast in their own projects** — not for
hacking on this repository.

After `composer require johnnyjoy/fast`, copy or reference these files in your
app so your agent knows how to use `\Fast` correctly.

| Skill | Path |
|-------|------|
| Using Fast | [`fast/SKILL.md`](fast/SKILL.md) |

**In Cursor:** attach `@skills/fast/SKILL.md` from your project (copy the
`skills/` folder from `vendor/johnnyjoy/fast/skills/`, or symlink it).

**On GitHub:** [skills/fast/SKILL.md](fast/SKILL.md)
